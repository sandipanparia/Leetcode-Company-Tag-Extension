This extension adds company tags and interview insights directly on LeetCode problem pages. It helps users quickly understand which companies frequently ask a problem and how important the problem is in real interviews.

🚀 Key Features
🏢 Shows company names (Google, Amazon, Meta, Microsoft, etc.) on each problem
📊 Displays frequency / popularity score of problems in interviews
🔍 Helps identify high-priority interview questions
⚡ Lightweight UI integrated directly into LeetCode
📈 Optional sorting/filtering by company or difficulty
💡 Helps in FAANG + product-based company preparation
